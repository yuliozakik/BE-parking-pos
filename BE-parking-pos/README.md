Parking POS - Spring Boot Backend
--------------------------------
Requirements:
- Java 21
- Maven 3.8+
- MySQL running and accessible

How to start:
1. Create database: see sql/init.sql
2. Edit src/main/resources/application.properties with DB credentials
3. Build: mvn clean package
4. Run: java -jar target/parking-pos-0.0.1-SNAPSHOT.jar
